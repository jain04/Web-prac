Selectors in html

Element Selector

The element selects HTML elements based on the name.

Id Selector

The id selector uses the id atrribute of an HTML element to select to a specific element.The id of the element is unique within the page, so the id selector is used to select one unique element!

Class Selector

The class selector selector selects HTML elements with a specific class attribute.To select elements with a specific class, write a period() chacter,followed by the class name.
